# ⚡ Быстрое исправление проблем с Telegram

## Что проверить прямо сейчас

### 1. URL в BotFather
- Откройте [@BotFather](https://t.me/botfather)
- `/myapps` → ваше приложение
- Проверьте Web App URL
- Должен быть: `https://your-app.netlify.app` (или vercel.app)
- **НЕ должен быть:** `http://` или `ngrok`

### 2. Работает ли сайт в браузере?
- Откройте URL от Netlify в браузере на компьютере
- Должна открыться страница приложения
- Если не открывается - проблема с деплоем

### 3. Работает ли backend?
- Откройте: `https://your-backend.railway.app/health`
- Должно показать: `{"status":"ok"}`
- Если не работает - проблема с backend

### 4. Какие ошибки видите?
- В Telegram: долгое нажатие на приложение → "Inspect Element"
- Или откройте URL в браузере на телефоне
- F12 → Console
- Какие ошибки?

## Частые проблемы

### Белый экран
→ Проверьте консоль браузера на ошибки

### "Failed to load"
→ Проверьте, что backend доступен

### "CORS error"
→ Я уже исправил CORS в backend, перезапустите backend

### Приложение не открывается
→ Проверьте URL в BotFather - должен быть HTTPS

Если backend на Railway:
