## Что нужно проверить

### 1. Backend доступен?

1. Откройте backend URL:
   - `https://your-backend.railway.app/health`
2. Должно показать: `{"status":"ok"}`
3. Если не работает - проблема с backend

### 2. Секрет VITE_API_URL правильный?

1. GitHub → **Settings** → **Secrets and variables** → **Actions**
2. Проверьте секрет `VITE_API_URL`:
   - ✅ Должен быть добавлен
   - ✅ Значение должно быть правильным
   - ✅ Должен заканчиваться на `/api` (например: `https://your-backend.railway.app/api`)

### 3. Перезапустите workflow

После проверки/исправления:

1. **Actions** → **Deploy to GitHub Pages**
2. Нажмите **"Run workflow"**
3. Дождитесь завершения
4. Проверьте сайт снова
