# ✅ Проверка работы в Telegram

## Что проверить

### 1. URL в BotFather
```
1. Откройте @BotFather
2. /myapps
3. Выберите ваше приложение
4. Проверьте Web App URL
```

**Должно быть:**
- ✅ `https://your-app.netlify.app` (или vercel.app)
- ✅ Начинается с `https://`
- ❌ НЕ должно быть `http://` или `ngrok`

---

### 2. Проверка в браузере

**На компьютере:**
1. Откройте URL от Netlify в браузере
2. Должна открыться страница приложения
3. Если не открывается → проблема с деплоем

**На телефоне:**
1. Откройте URL в браузере на телефоне
2. Должна открыться страница приложения
3. Если не открывается → проблема с деплоем или сетью

---

### 3. Проверка backend

1. Откройте: `https://your-backend.railway.app/health`
2. Должно показать: `{"status":"ok"}`
3. Если не работает → проблема с backend

---

### 4. Проверка в Telegram

1. Откройте бота в Telegram
2. Нажмите кнопку "Open App" или "Открыть приложение"
3. Должно открыться приложение

**Если не открывается:**
- Проверьте URL в BotFather
- Проверьте, что сайт работает в браузере
- Проверьте консоль (если доступна)
